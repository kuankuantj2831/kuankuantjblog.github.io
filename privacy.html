🔒 隐私政策

最后更新：2026 年 3 月 22 日

1. 概述

「Hakimi 的猫爬架」（以下简称"本站"）重视你的隐私。本政策说明我们如何收集、使用、存储和保护你的个人信息。使用本站即表示你同意本政策。

2. 我们收集的信息

信息类型具体内容收集方式
账号信息用户名、邮箱地址、加密后的密码注册时提供
个人资料头像、个人简介你主动设置
内容数据发布的文章、评论、点赞记录使用过程中产生
虚拟物品硬币余额、等级经验使用过程中产生
访问数据访问时间、页面浏览量自动收集

我们不会收集你的真实姓名、身份证号、手机号、地理位置或支付信息。

3. 信息使用目的

我们收集的信息仅用于：

  • 提供和维护本站的核心功能（登录、发文、评论、等级系统等）。
  • 发送与你账号相关的通知（如被评论、被点赞提醒）。
  • 统计网站访问数据以改善用户体验。
  • 防止滥用行为和保障网站安全。

我们不会将你的信息用于广告投放或出售给第三方。

4. 信息存储与安全

  • 密码安全：你的密码经过 bcrypt 加密存储，即使数据库泄露也无法还原明文密码。
  • 传输加密：本站全站使用 HTTPS 加密传输，防止数据被窃听。
  • 身份验证：使用 JWT（JSON Web Token）进行身份验证，Token 有时效限制。
  • 人机验证：注册和登录使用 Cloudflare Turnstile 防止机器人攻击。
  • 数据存储：数据存储在腾讯云服务器上，受腾讯云安全体系保护。

5. 第三方服务

本站使用以下第三方服务，它们可能会收集部分匿名数据：

  • Cloudflare — CDN 加速和安全防护，可能收集 IP 地址用于安全分析。
  • GitHub Pages — 静态页面托管服务。
  • 腾讯云 SCF — 后端 API 服务托管。
  • 腾讯云 COS — 用户头像等文件存储。

这些第三方服务有各自的隐私政策，我们建议你查阅了解。

6. Cookie 与本地存储

  • 本站使用 localStorage 存储登录 Token 和用户偏好设置（如主题模式）。
  • Cloudflare 可能设置安全相关的 Cookie。
  • 我们不使用追踪型 Cookie 或第三方广告 Cookie。

7. 你的权利

你对自己的个人信息享有以下权利：

  • 查看 — 在个人中心查看你的所有账号信息。
  • 修改 — 随时修改你的用户名、头像和个人简介。
  • 删除 — 你可以联系站长申请删除账号及所有相关数据。
  • 导出 — 你可以联系站长申请导出你的个人数据。

8. 未成年人保护

本站不面向 13 岁以下的儿童。如果我们发现不慎收集了 13 岁以下儿童的信息，将立即删除。如果你是未成年人，请在监护人的指导下使用本站。

9. 政策变更

我们可能会不时更新本隐私政策。更新后的政策将在本页面公布并标注更新日期。重大变更时，我们会通过站内通知提醒你。

10. 联系我们
