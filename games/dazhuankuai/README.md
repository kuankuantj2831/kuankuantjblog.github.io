# 打砖块游戏 (Brick Breaker)

一个使用 HTML5 Canvas 和 JavaScript 构建的经典打砖块游戏，具有现代UI、音效和粒子特效。

## 功能特点

- 🎮 **经典玩法**：使用挡板反弹小球击碎所有砖块
- 🌈 **精美视觉效果**：渐变背景、粒子特效、动态砖块颜色
- 🔊 **音效系统**：碰撞、击碎、升级、游戏结束等音效
- ⚡ **特殊砖块**：随机出现具有特殊效果的砖块（额外生命、加速、挡板变大）
- 📊 **游戏统计**：实时显示生命值、分数、等级和速度
- 🎛️ **可调设置**：音量控制、难度调节
- 🎯 **响应式设计**：适配不同屏幕尺寸
- ⌨️ **键盘控制**：支持方向键、A/D键、空格键和P键

## 如何运行

1. **直接打开**：双击 `index.html` 在浏览器中打开
2. **本地服务器**：使用Python或其他HTTP服务器
   ```bash
   python -m http.server 8080
   ```
   然后在浏览器访问 `http://localhost:8080`

## 游戏控制

- **左箭头 / A**：挡板向左移动
- **右箭头 / D**：挡板向右移动
- **空格键**：发射小球 / 暂停游戏
- **P键**：暂停 / 继续游戏
- **鼠标点击**：开始/重新开始游戏

## 游戏规则

1. 使用挡板反弹小球，击碎所有砖块
2. 每个砖块得10分，特殊砖块额外加分
3. 不要让小球掉落到挡板下方，否则损失一条生命
4. 清除所有砖块进入下一关，速度会逐渐增加
5. 特殊砖块效果：
   - ❤️ 额外生命：增加一条生命
   - ⚡ 加速：暂时提高小球速度
   - ⬆️ 挡板变大：暂时增大挡板宽度

## 开发技术

- **HTML5 Canvas**：游戏渲染
- **CSS3**：渐变、动画、响应式布局
- **JavaScript (ES6)**：游戏逻辑、碰撞检测、状态管理
- **Font Awesome**：图标
- **Google Fonts**：字体

## 自定义和扩展

### 修改游戏参数
在 `game.js` 文件中可以调整以下常量：
```javascript
const CANVAS_WIDTH = 800;      // 画布宽度
const CANVAS_HEIGHT = 500;     // 画布高度
const PADDLE_WIDTH = 100;      // 挡板宽度
const BALL_RADIUS = 10;        // 小球半径
const BRICK_ROWS = 6;          // 砖块行数
const BRICK_COLUMNS = 10;      // 砖块列数
```

### 添加新音效
1. 将音效文件放入 `assets/sounds/` 目录
2. 在 `index.html` 中添加 `<audio>` 元素
3. 在 `game.js` 中引用并播放

### 修改样式
在 `style.css` 中可以调整颜色、字体和布局。

## 已知问题

- 音效在某些浏览器中可能需要用户交互后才能播放
- 游戏在高刷新率屏幕上可能运行过快（可通过调整 `requestAnimationFrame` 解决）
