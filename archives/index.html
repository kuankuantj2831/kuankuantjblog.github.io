如何添加新文章和附件

由于这是一个静态网站，您需要通过以下步骤在本地添加新内容，然后推送到GitHub以更新网站。

第一步：创建新文章文件

您的网站似乎是由Hexo生成的。要创建一篇新文章，请在您的项目根目录打开终端并运行以下命令：

hexo new "您的文章标题"

这将在 source/_posts 文件夹下创建一个新的Markdown文件（例如 `您的文章标题.md`）。

第二步：撰写文章

使用Markdown编辑器打开新创建的文件并开始写作。您可以在文件顶部的"front-matter"部分设置标题、日期、标签等信息。

第三步：添加附件（如图片）

  1. 将您的图片或其他附件文件放入项目中的 source/images/ 文件夹。
  2. 在您的Markdown文章中，使用以下语法引用图片：
    ![图片描述](/images/您的图片文件名.jpg)

第四步：本地预览和部署

  1. 在终端中运行 hexo server 来本地预览您的网站，确保一切显示正常。
  2. 确认无误后，运行 hexo deploy 或使用以下Git命令来部署您的更改：
git add .
git commit -m "Add new post: 您的文章标题"
git push origin main

几分钟后，您的新文章和附件就会显示在网站上。
